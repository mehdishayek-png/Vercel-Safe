# Midas Match — Impeccable Design Context

## Target Audience
Job seekers in India (primarily tech/operations professionals aged 22-35) who want AI-powered job matching that goes beyond basic keyword search. They're tech-savvy but time-constrained.

## Use Cases
- Upload resume and get AI-matched job listings from 8+ sources
- Deep analysis of individual job fit with salary estimates
- Track saved jobs and applications in one place
- Generate cover letters and interview prep

## Brand Personality / Tone
**Refined, confident, and modern** — not flashy or gimmicky. Think "premium fintech" meets "smart career tool." Clean surfaces, purposeful whitespace, subtle depth. The gold accent (#D4AF37) nods to the "Midas touch" metaphor without being gaudy.

## Design System
- **Framework**: Next.js + Tailwind CSS
- **Dark mode**: Class-based (`dark:` prefix), tokens: `dark:bg-[#0f1117]` (app), `dark:bg-[#1a1d27]` (card), `dark:bg-[#22252f]` (hover), `dark:border-[#2d3140]`
- **Brand colors**: `brand-600` (primary gold), `accent-600` (secondary)
- **Typography**: System font stack via Tailwind defaults
- **Components**: Custom (no UI library), lucide-react icons, framer-motion animations
- **Accessibility**: WCAG 2.2 AA target

## Differentiation
The "Panda Score" — a multi-signal matching algorithm that scores jobs on 7+ dimensions rather than just keywords. The interface reveals this intelligence through match rings, heuristic breakdowns, and confidence indicators.
